# API для продавца часов

## seller.py

Данный скрипт получает артикулы товаров магазина OZON с помощью API-запросов, затем скачивает остатки с сайта Casio и формирует спискок актуальных к продаже часов. Так же с помощью этого списка обновляет выставленые на маркетплейсе часы и цену на них.

## market.py

Данный скрипт запрашивает артикулы товаров Яндекс-маркет, проверяет актуальность продаваемых товаров и их цены, затем обновляет их на самом маркетплейсе. Cначала для модели FBS (Fulfillment by Seller), затем аналогично для DBS (Delivery by Seller).
